## Install

```sh
git clone https://github.com/IhsanDevs/LaravelYoutubeDownloader && cd LaravelYoutubeDownloader
```

```sh
cp .env.example .env
```

Set the configuration in the `.env` file.

```sh
npm install && npm install --save turbolinks && npm run dev
```

## Run tests

```sh
php artisan serve
```
